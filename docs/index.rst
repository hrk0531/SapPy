Introduction
============

sappy is a Python library designed to simplify statistical analysis. This documentation provides information on how to install the library, basic usage, and details about the available modules.

Features
========

- Data preprocessing capabilities
- Implementation of various statistical analysis methods
- Visualization tools
- Customizable analysis pipelines

Installation
============

Installing sappy is straightforward. Use the following command to install:

.. code-block:: bash

    pip install sappy
